# IMDB_mid
This is the midterm project for IMDB.

### Requirement
node version >= v.16.9.1
npm version >= v7.21.1
yarn version >= v1.22.10

### Usage

1. Clone this file
```shell
git clone https://github.com/Chiuweiyu/IMDB_mid.git
```
2. Go to the `backend` and `frontend` dictionaries to start the project.
